# タスク: iPhone向けUI最適化（ノッチ対応・スライダー改善）

iPhoneでの利用体験を向上させるため、以下の2点の修正を行います。

## 1. ノッチ（セーフエリア）対応
- iPhoneの上部ノッチ部分にタイトルが被る問題を解消します。
- `viewport-fit=cover` の設定と、`env(safe-area-inset-top)` を使用したパディングの調整を行います。

## 2. スライダー操作性の向上
- 焦点距離、F値、フォーカス距離のスライダーが小さく操作しづらいため、タッチターゲットを拡大します。
- スライダーのつまみ（thumb）とトラックのサイズを大きくし、モバイルでの操作性を改善します。
