# タスク: iPhone横向きレイアウト対応・ノッチはみ出し修正

1. **ノッチはみ出し（パンチホール被り）の修正**
   以前の修正で `env(safe-area-inset-top)` を使用しましたが、CapacitorでビルドしたiOSネイティブアプリ環境では正しく機能していない可能性があります。
   `pl-safe`, `pr-safe`, `pt-safe` などのユーティリティを横持ち時にも対応できるように拡張し、さらに Capacitor/iOS プラットフォームを考慮した調整を行います。

2. **モバイル端末の横向き（Landscape）レイアウトの改善**
   現在、スマートフォンを横に向けた際にビューアー（カメラビュー/トップビュー）が画面の大部分を占有し、パラメーターパネルにアクセスできない問題があります。
   横向きにされた場合、モバイルであっても画面を2カラム（左1/3にパネル、右2/3にビューアー）に分割するPCのようなレイアウトに変更します。
