# Task: カメラ画角シミュレーターのReact + Capacitorアプリ化

## 概要
現在HTML/JavaScript/Three.jsで実装されている「カメラ画角＆圧縮効果シミュレーター」を、モバイルアプリ化（iOS/Android対応）を見据えて、Reactベースのモダンなアーキテクチャに完全移行する。

## 目標
1. **React(Vite) + TypeScript環境の構築**:  モダンなフロントエンド開発環境をセットアップする。
2. **React Three Fiber (R3F) の導入**: Three.jsの操作をReactの宣言的な記法に置き換え、コードの見通しと保守性を劇的に向上させる。
3. **Tailwind CSSによるUIのリファクタリング**: 既存のUIを「より美しく、プレミアムなデザイン」にブラッシュアップし、スマホライクなコンポーネントを設計する。
4. **Capacitorの導入**: WebアプリをiOS/Androidアプリとしてパッケージングするための枠組みを追加する。

## 完了条件
- Vite(React+TS)環境がローカルで正常に起動すること。
- 既存のすべてのシミュレーション機能（画角計算、センサーサイズ変更、アスペクト比クロップなど）がReact上で再現されていること。
- React Three Fiber経由での3D描画が正しく機能していること。
- Capacitorの初期設定が完了し、iOS/Android向けビルドの準備が整うこと。
